## 🔧 API ENDPOINTS

```
GET  /health
     └─ Server health check

POST /detect/image
     └─ Detect objects in uploaded image

GET  /detect/video/stream?source=0
     └─ MJPEG video stream

WebSocket /ws/video?source=0
     └─ Real-time WebSocket stream

POST /detect/video/file
     └─ Process uploaded video file

GET  /models/classes
     └─ Get detection classes

GET  /config
     └─ API configuration
```
